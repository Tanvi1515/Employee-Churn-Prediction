# 📊 Employee Churn Prediction using Machine Learning

## 📌 Project Overview

Employee attrition is a major challenge for organizations. This project predicts whether an employee will leave the company using machine learning algorithms.

---

## 🎯 Objective

To build a classification model that predicts employee attrition and helps organizations take proactive decisions.

---

## 📂 Dataset

* IBM HR Analytics Employee Attrition Dataset
* Contains employee details like age, salary, job role, etc.

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## 🤖 Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Naive Bayes
* Support Vector Machine (SVM)

---

## 📊 Data Preprocessing

* Removed unnecessary columns
* Encoded categorical variables
* Feature scaling using StandardScaler
* Train-test split (80-20)

---

## 🏆 Model Performance

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 89.46%      |
| Decision Tree       | 78.91%      |
| Naive Bayes         | 84.35%      |
| SVM                 | 88.78%      |

---

## 🔍 Key Insights

* Employees with low income are more likely to leave
* Overtime strongly affects attrition
* Job role impacts employee retention
